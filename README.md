# Анализ продаж супермаркета (Superstore)

##  Описание проекта
Проект посвящен анализу данных о продажах супермаркета.  
**Цель** — выявить прибыльные и убыточные категории товаров, оценить влияние скидок на прибыль, определить самых ценных клиентов.

##  Используемые инструменты
- **SQLite** — написание аналитических запросов
- **Power BI** — создание интерактивного дашборда
- **Git** — контроль версий и портфолио

##  Ключевые выводы
1. **Самая прибыльная категория** — `Technology` (145 454 прибыли).
2. **Влияние скидок:** скидка >20% делает заказы убыточными (при 30% скидки средняя прибыль становится отрицательной).
3. **Убыточные подкатегории:** `Tables`, `Bookcases`, `Supplies`. Рекомендуется пересмотреть цены или маркетинговую стратегию.
4. **Топ-10 товаров** дают более 20% всей прибыли компании.
